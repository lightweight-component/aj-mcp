# AJ-MCP Common

Common library for AJ-MCP, it contains the implementation of the Model Context Protocol and some other common classes.

## Install

Runs on Java8+. Maven:

```xml

<dependency>
    <groupId>com.ajaxjs</groupId>
    <artifactId>aj-mcp-common</artifactId>
    <version>1.5</version>
</dependency>
```
